### Installation

- `git clone git@github.com:vuchkov/laranuxtshop.git`
- `cd laranuxtshop`
- `pnpm i`
- `composer install`
- `cp .env.example .env`

### Local Environment

- run `pnpm run dev` in one terminal for our nuxt dev setup.
- run `pnpm run api` (alias for `./artisan serve`) in another terminal for 
our laravel API
